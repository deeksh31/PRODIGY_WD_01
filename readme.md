# Responsive Landing Page

## Overview

This project is a responsive landing page created using HTML, CSS and JavaScript.

## Features

- Responsive Design
- Fixed Navigation Bar
- Navbar Color Change on Scroll
- Hover Effects
- Smooth Scrolling
- Hero Section
- About Section
- Services Section
- Contact Form

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download the project.
2. Open the folder.
3. Open index.html in your browser.
